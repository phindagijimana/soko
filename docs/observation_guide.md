# Observation Guide

## Purpose
Observe behavior, not preferences. The main question is whether the system helps real trade happen.

## What to watch
- How many farmers create a listing without help?
- How many buyers reach the order step?
- How long does it take from listing to first order?
- How often do users abandon the flow?
- Which fields confuse users?
- How often do SMS notifications matter to the next action?

## Simple daily metrics
- New farmers onboarded
- New buyers onboarded
- Listings created
- Orders placed
- Orders accepted
- Orders completed
- Reviews submitted

## Signs of traction
- Users come back the next day without being pushed
- Buyers ask for more listings
- Farmers ask how to post the next harvest
- Orders move from pending to accepted to completed

## Signs you should fix something before scaling
- Users need your help on every action
- OTP or SMS creates confusion
- Orders stay pending too long
- Crop quantities and prices are entered inconsistently
